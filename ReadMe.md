# TechX 2026 - College Tech Fest Website

A multi-page college tech fest website built using only HTML.

## Features

- Multi-page website
- Semantic HTML
- Registration Form
- Event Schedule
- Gallery
- Contact Page
- Google Maps
- Audio & Video
- Accessible Forms
- Tables
- Figures
- Details & Summary

## Technologies Used

- HTML5

## Purpose

This project was created to practice HTML fundamentals before learning CSS and JavaScript.
