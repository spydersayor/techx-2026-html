# 🚀 TechX 2026 – College Tech Fest Website

A modern multi-page **College Tech Fest Website** built as part of my **Project Vishkarma** learning journey.

This project started as an **HTML-only website** and is being continuously improved as I learn new web development technologies.

---

## 📌 Project Overview

TechX 2026 is a fictional college technical festival website that provides visitors with information about events, registration, gallery, and contact details.

The primary goal of this project is to practice **real-world frontend development** while following professional software development workflows using **Git** and **GitHub**.
